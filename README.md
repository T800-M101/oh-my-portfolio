# OhMyPortfolio

A modern, responsive, and themeable portfolio built with **Angular 18** and **Signals**, designed to showcase your projects, skills, and experience with style.


This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 18.2.19.

## 🚀 Features

- 🌙 **Dark Mode** toggle with saved preference
- 📱 Fully responsive design for all devices
- 🧠 Built with Angular 18 and the new **Signals API**
- 🗂️ Modular structure with reusable components
- 🌍 Language support (EN/ES toggle ready)
- 📂 Project data and user profile managed with a service
- ⚙️ Smooth animations and modern UI using CSS variables

## 🧰 Setup & Installation


## Clone the repository
git clone https://github.com/T800-M101/oh-my-portfolio.git
cd oh-my-portfolio

# Install dependencies
npm install

# Run the app locally
ng serve
